# 📢 Online Notice Board System

An Online Notice Board System built using **Django** that enables institutions to publish, manage, and distribute notices digitally. The platform provides a dedicated admin panel for managing notices while allowing users to browse notices by department or category, search and filter announcements, subscribe for email notifications, and download attached documents.


## ✨ Features


### 👤 User Features
- View published notices
- Search notices by title or keyword
- Filter notices by month and year
- Browse notices by department and category
- View notice attachments (PDF, images, Word, Excel, etc.)
- Subscribe to departments/categories via email
- Receive email notifications for new notices and major updates
- "Updated" badge for modified notices


### 🛠️ Admin Features
- Secure admin authentication
- Dashboard with notice statistics
- Create, edit, publish, schedule, and delete notices
- Upload attachments (PDF, Images, DOCX, XLSX, etc.)
- Manage departments and categories
- Manage email subscriptions
- Schedule notices for automatic publishing
- Super Admin approval for new administrators


## 🏗️ Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **Email Service:** Django SMTP
- **Version Control:** Git & GitHub
